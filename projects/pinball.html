Pinball Machine — Distributed Embedded Control

C++ArduinoI2CEmbeddedFSMCNC Machining

Full machine prototype showing the custom playfield, flippers, bumpers, and turntable assembly.

Overview

Designed and built a fully functional pinball machine from scratch — mechanical fabrication, circuit design, embedded firmware, and game logic. The machine was CNC-machined from custom enclosure designs and implemented a distributed embedded control architecture.

Rather than running everything from a single microcontroller, each physical actuator (bumper, paddle) had its own dedicated Arduino Nano, communicating with a central controller over I2C. This kept actuator response tight and local while centralizing game state.

Technical Highlights
  • CNC-machined enclosure and mechanical components from custom designs
  • Distributed architecture: independent Arduino Nanos per actuator (bumpers and paddles)
  • I2C bus connecting peripheral nodes to a central controller Arduino
  • Game logic and scoring implemented in C++ using a finite state machine architecture
  • Safety interlocks and interrupt-driven I/O for reliable actuator response
  • Debugged bus contention, I2C timing races, and shared-state issues across networked microcontrollers
