Raw IQ time series, amplitude envelope, and constellation extracted from the PlutoSDR report

Overview

Reverse-engineered an unknown wireless transmission starting from raw signal capture, working through every layer of the physical layer receiver chain to recover the original bitstream.

The challenge was operating without any prior knowledge of the signal — modulation scheme, symbol rate, carrier frequency offset, and framing structure all had to be determined from the captured IQ samples using MATLAB and GNU Radio.

Receiver Pipeline
01Raw signal capture via PlutoSDR hardware
02Modulation scheme identification from signal characteristics
03Timing synchronization — symbol timing recovery
04Phase synchronization — carrier phase offset correction
05Carrier frequency synchronization — CFO estimation and correction
06Preamble detection and frame alignment
07Bitstream recovery and decoding
08BER analysis to validate receiver performance
